About

Grubbernaut is designed to assist a user explore the tastiest, most sanitary food in Wake County so that you can decide if you want to grub or not. Use our search feature to search for a restaurant, or browse to explore the restaurants with the most recent inspections.

About The Project

This project was developed out of a desire to make public information that is not easily accessible available in a user-friendly format.

To create the Grubbernaut app, data from four API's were used: three from Wake County Health Inspections and the Yelp Fusion API.

Grubbernaut was built using Django, JavaScript, HTML and CSS. Graphic elements were developed using Charts.js.

Grubbernaut's clean and simple design makes comparing health inspection scores and Yelp ratings quick and easy for the user.

APIs Used:

Wake County Restaurants API
Wake County Inspections API
Wake County Violations API
Yelp Fusion API
